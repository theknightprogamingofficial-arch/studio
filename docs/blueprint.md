# **App Name**: LeafWise

## Core Features:

- Image Upload/Capture: Allows users to upload plant images from their device or capture them using their camera.
- AI Plant Identification: Uses Google's Gemini Vision model to analyze plant images and identify the species.
- Detailed Plant Information: Presents a results card with the plant's image, common name, and a fun fact.
- Save to My Garden: Allows users to save identified plants to their personal virtual garden.
- Track Plant Care: Tracks the last watering date for each plant and allows users to update it.
- Smart Watering Predictions: Uses AI to predict when a plant should be watered next, based on its species and last watering date. The AI will use care guides as a tool for this.
- AI Plant Doctor: Offers a conversational AI that diagnoses plant problems and provides care advice based on the specific plant. The AI will use care guides as a tool for this.
- Mobile First Design: Focus on mobile first app.
- Bottom Navigation Dock: Doc at the bottom to access different features.
- Easy to Use: Easy to use.
- React Native: Made with React Native

## Style Guidelines:

- Primary color: Soft Green (#A7D1AB) to evoke nature and growth.
- Background color: Very light beige (#F5F5DC) for a natural, earthy feel.
- Accent color: Muted orange (#343A40$) for highlights and call-to-action buttons, providing contrast and warmth.
- Body: 'PT Sans', a humanist sans-serif font for body text to offer readability with a touch of warmth.
- Headlines: 'Playfair', a modern serif font that would match PT Sans and give the app a fashionable feel.
- Use clear, nature-inspired icons for plant categories, watering reminders, and other actions.
- Clean, card-based layout for plant display, with a side panel for garden overview.
- Subtle transitions and animations for watering updates and AI diagnosis.
- Modern and Clean design.
